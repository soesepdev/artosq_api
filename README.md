# artosq

API backend untuk aplikasi artosq, dibangun dengan Express.js dan TypeScript, menggunakan PostgreSQL sebagai database.

## Features

- RESTful API
- Modular structure
- PostgreSQL database
- TypeScript support
- Environment configuration

## Installation

```bash
git clone https://github.com/soesepdev/artosq_api.git
cd artosq_api
npm install
